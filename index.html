How to Access Your Apple Calendar

Method 1: Export ICS File

  1. Open Apple Calendar on your Mac
  2. Select the calendar you want to export
  3. Go to File → Export → Export...
  4. Save the .ics file
  5. Import it using the button above

Method 2: CalDAV Connection

  1. Go to appleid.apple.com
  2. Sign in and go to Security → App-Specific Passwords
  3. Generate a password for "Calendar Viewer"
  4. Enter your Apple ID and the generated password above
